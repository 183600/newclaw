# `newclaw tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/tui)

## Examples

```bash
newclaw tui
newclaw tui --url ws://127.0.0.1:18789 --token <token>
newclaw tui --session main --deliver
```

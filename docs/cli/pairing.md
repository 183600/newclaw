# `newclaw pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:

- Pairing flow: [Pairing](/start/pairing)

## Commands

```bash
newclaw pairing list whatsapp
newclaw pairing approve whatsapp <code> --notify
```
